## ✨ Key Features

### 🎨 Modern UI/UX
- **Parallax Hero Section:** Interactive text layers that move at different speeds on scroll.
- **Bento Grid Layout:** A modern, responsive grid for the "About Me" section (inspired by Apple/Linear design).
- **Sticky Card Stack:** "Work" section features projects that stack on top of each other while scrolling.
- **Infinite Marquee:** Smooth, continuous loop for testimonials with CSS masking for seamless blending.
- **Glassmorphism:** Premium frosted glass effects used in navigation and cards.

### ⚡ Performance & Optimization
- **60 FPS Animations:** Powered by **Framer Motion** with GPU acceleration (`will-change`, `transform`).
- **Smooth Scrolling:** Integrated **Lenis** for buttery smooth scroll inertia.
- **Mobile First:** Fully optimized for touch devices (no hover stickiness, correct touch targets, prevented horizontal overflow).
- **Lazy Loading:** Images and heavy components (like the CMS modal) are lazy-loaded to ensure instant initial paint.
- **Asset Optimization:** All assets converted to **WebP** for minimal file size.

### 🛠️ Dynamic Content (CMS)
- **Sanity.io Integration:** A "Work Log" feature that allows posting updates, thoughts, and quick wins directly from a headless CMS without redeploying the site.
- **Rich Text Support:** Renders images, links, and formatted text dynamically.
